        <li>HTML 文档包含了HTML 标签及文本内容。</li>
        <li>HTML 标签是由尖括号包围的关键词，比如 <xmp><html></xmp><ol>
        <li>HTML 标签通常是成对出现的，比如 <xmp><b></xmp> 和 <xmp></b></xmp></li>
